FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for porfolio building

Is Frontend Mentor free?

Yes, Frontend Mentor is completely free to use.

Can i use Frontend Mentor projects in my portfolio?

Yes, you can use any projects you create for your portfolio.

How can I get help if i'm stuck on a challenge?

You can ask for help in the community forums or watch the solution videos.
